## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Ensure the dataset is available:
```bash
# Dataset should be at data/data.csv
# If not, run: python download_dataset.py
```

3. Ensure visualization images exist:
   - `ga_flowchart.png`
   - `pso_flowchart.png`
   - `convergence_comparison.png`
   - `results_comparison.png`

## Running the Application

```bash
streamlit run streamlit_app.py
```

The application will open in your default web browser at `http://localhost:8501`

## Requirements

- Python 3.7+
- Streamlit 1.28.0+
- All dependencies from requirements.txt
- Dataset at `data/data.csv`
- Visualization images (generated by running main.py)

## Notes

- The application uses session state to store results between page navigations
- Algorithm execution may take a few seconds depending on parameters
- All visualizations are interactive and responsive
